    Worksheet Gaussian elimination
    

    MTH 204: January 15
    🔗
    In this activity, we will develop an algorithm that transforms a system of linear equations into a specific form, called reduced row echelon form, that allows us to instantly draw conclusions about the set of solutions.
    🔗

    1.

    Suppose we have a linear system whose augmented matrix is
    \begin{equation*} \left[ \begin{array}{ccc|c} 1 \amp 0 \amp 0 \amp -2 \\ 0 \amp 1 \amp 0 \amp 4 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array} \right] \end{equation*}
    What does this tell us about the set of solutions to the linear system? Is there 0, 1, or infinitely many solutions?
    🔗
    Solution.
    If the variables are \(x\text{,}\) \(y\text{,}\) and \(z\text{,}\) we know that \(x=-2\text{,}\) \(y=4\text{,}\) and \(z=1\text{.}\) Therefore, there is exactly one solution.
    🔗
    🔗
    🔗

    2.

    Let’s consider this linear system.
    \begin{equation*} \begin{alignedat}{4} x \amp {}+{} \amp y \amp {}+{} \amp 2z \amp {}={} \amp 1 \\ 2x \amp {}-{} \amp y \amp {}-{} \amp 2z \amp {}={} \amp 2 \\ -x \amp {}+{} \amp y \amp {}+{} \amp z \amp {}={} \amp 0 \\ \end{alignedat} \end{equation*}
    🔗

    (a)

    Write this system as an augmented matrix.
    🔗
    Solution.
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 1 \amp 2 \amp 1 \\ 2 \amp -1 \amp -2 \amp 2 \\ -1 \amp 1 \amp 1 \amp 0 \\ \end{array}\right] \end{equation*}
    🔗
    🔗
    🔗

    (b)

    For the first step in the Gaussian elimination algorithm, we would like to make the entry in the second row and first column a “0”. You can do this by performing a row replacement operation, multiplying row 1 by -2 and adding it to row 2. Sometimes, people write this as \(-2R_1 + R_2 \to R_2\text{.}\)
    🔗
    Solution.
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 1 \amp 2 \amp 1 \\ 0 \amp -3 \amp -6 \amp 0 \\ -1 \amp 1 \amp 1 \amp 0 \\ \end{array}\right] \end{equation*}
    🔗
    🔗
    🔗

    (c)

    For the second step, we would like to make the entry in the third row and first column a “0”. Determine the row replacement operation you would use and then perform that operation.
    🔗
    Solution.
    We can use \(R_1+R_3\to R_3\) and obtain
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 1 \amp 2 \amp 1 \\ 0 \amp -3 \amp -6 \amp 0 \\ 0 \amp 2 \amp 3 \amp 1 \\ \end{array}\right] \end{equation*}
    🔗
    🔗
    🔗

    (d)

    OK, good. We have essentially eliminated \(x\) from the second and third equation. You have some choice in what comes next, but it might be convenient to make the entry in the second row, second column a “1”. How would you scale the second row (i.e. multiply it by a nonzero number) to produce a “1” in that position?
    🔗
    Solution.
    Scaling the second row by \(-1/3\) gives
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 1 \amp 2 \amp 1 \\ 0 \amp 1 \amp 2 \amp 0 \\ 0 \amp 2 \amp 3 \amp 1 \\ \end{array}\right] \end{equation*}
    🔗
    🔗
    🔗

    (e)

    The next step would be to produce a “0” in the third row, second column. How can you do that?
    🔗
    Solution.
    We perform \(-2R_2+R_3\to R_3\) so that
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 1 \amp 2 \amp 1 \\ 0 \amp 1 \amp 2 \amp 0 \\ 0 \amp 0 \amp -1 \amp 1 \\ \end{array}\right] \end{equation*}
    🔗
    🔗
    🔗

    (f)

    From this point, how can you finish the Gaussian elimination algorithm to find the solution to the original linear system? Be specific about naming each step of the algorithm.
    🔗
    Solution.
    \begin{align*} \left[\begin{array}{rrr|r} 1 \amp 1 \amp 2 \amp 1 \\ 0 \amp 1 \amp 2 \amp 0 \\ 0 \amp 0 \amp -1 \amp 1 \\ \end{array}\right] \amp \sim \left[\begin{array}{rrr|r} 1 \amp 1 \amp 2 \amp 1 \\ 0 \amp 1 \amp 2 \amp 0 \\ 0 \amp 0 \amp 1 \amp -1 \\ \end{array}\right]\\ \amp \sim \left[\begin{array}{rrr|r} 1 \amp 1 \amp 2 \amp 1 \\ 0 \amp 1 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \amp -1 \\ \end{array}\right]\\ \amp \sim \left[\begin{array}{rrr|r} 1 \amp 1 \amp 0 \amp 3 \\ 0 \amp 1 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \amp -1 \\ \end{array}\right]\\ \amp \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \amp -1 \\ \end{array}\right] \end{align*}
    🔗
    🔗
    🔗

    (g)

    What can you say about the set of solutions to this linear system? Is there 0, 1, or infinitely many solutions?
    🔗
    Solution.
    There is exactly one solution: \(x=1\text{,}\) \(y=2\text{,}\) and \(z=-1\text{.}\)
    🔗
    🔗
    🔗
    🔗

    3.

    Rewrite the following linear system as an augmented matrix and find the solution using Gaussian elimination.
    \begin{equation*} \begin{alignedat}{4} x \amp {}+{} \amp 2y \amp {}{} \amp \amp {}={} \amp 2 \\ -x \amp {}{} \amp \amp {}-{} \amp 4z \amp {}={} \amp -6 \\ 2x \amp {}+{} \amp 4y \amp {}+{} \amp z \amp {}={} \amp 7 \\ \end{alignedat} \end{equation*}
    🔗
    Solution.
    We set up an augmented matrix and apply row operations so that
    \begin{align*} \left[\begin{array}{rrr|r} 1 \amp 2 \amp 0 \amp 2 \\ -1 \amp 0 \amp -4 \amp -6 \\ 2 \amp 4 \amp 1 \amp 7 \\ \end{array}\right] \amp \sim \left[\begin{array}{rrr|r} 1 \amp 2 \amp 0 \amp 2 \\ 0 \amp 2 \amp -4 \amp -4 \\ 2 \amp 4 \amp 1 \amp 7 \\ \end{array}\right]\\ \amp \sim \left[\begin{array}{rrr|r} 1 \amp 2 \amp 0 \amp 2 \\ 0 \amp 2 \amp -4 \amp -4 \\ 0 \amp 0 \amp 1 \amp 3 \\ \end{array}\right]\\ \amp \sim \left[\begin{array}{rrr|r} 1 \amp 2 \amp 0 \amp 2 \\ 0 \amp 1 \amp -2 \amp -2 \\ 0 \amp 0 \amp 1 \amp 3 \\ \end{array}\right]\\ \amp \sim \left[\begin{array}{rrr|r} 1 \amp 2 \amp 0 \amp 2 \\ 0 \amp 1 \amp 0 \amp 4 \\ 0 \amp 0 \amp 1 \amp 3 \\ \end{array}\right]\\ \amp \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp -6 \\ 0 \amp 1 \amp 0 \amp 4 \\ 0 \amp 0 \amp 1 \amp 3 \\ \end{array}\right] \end{align*}
    🔗
    🔗
    🔗

    4.

    We will make a few observations about situations that frequently occur.
    🔗

    (a)

    What does the equation \(0x+0y=0\) tell us, if anything, about the values of \(x\) and \(y\text{?}\)
    🔗
    Solution.
    It doesn’t tell us anything since any values of \(x\) and \(y\) cause the equation to be true.
    🔗
    🔗
    🔗

    (b)

    Suppose that you have a linear system in the variables \(x\) and \(y\) whose augmented matrix is as below. Describe the solution space in as much detail as possible.
    \begin{equation*} \left[ \begin{array}{rr|r} 1 \amp 0 \amp 3 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \\ \end{array} \right]. \end{equation*}
    🔗
    Solution.
    We can ignore the last row and note that \(x=3\) and \(y=0\text{.}\)
    🔗
    🔗
    🔗
    🔗

    5.

    Suppose that you have a linear system in the variables \(x\) and \(y\) whose augmented matrix is as below. Describe the solution space in as much detail as possible.
    \begin{equation*} \left[ \begin{array}{rr|r} 1 \amp 0 \amp 3 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array} \right]. \end{equation*}
    🔗
    Solution.
    The last row implies that \(0x+0y=1\text{.}\) However, there are no values for \(x\) and \(y\) that cause this equation to be true. This system therefore has no solutions.
    🔗
    🔗
    🔗

    6.

    The goal of Gaussian elimination is to create an augmented matrix that is in reduced row echelon form, which means
    • If the entries in a row are all zero, then the same is true of any row below it.
      🔗
    • If we move across a row from left to right, the first nonzero entry we encounter is 1. We call this entry the leading entry in the row.
      🔗
    • The leading entry in any row is to the right of the leading entries in all the rows above it.
      🔗
    • A leading entry is the only nonzero entry in its column.
      🔗
    Consider each of the matrices below and determine whether it is in reduced row echelon form or not. If not, perform a sequence of row operations to find a reduced row echelon form. Then use that form to determine whether the system has 0, 1, or infinitely many solutions.
    🔗

    (a)

    \begin{equation*} \left[ \begin{array}{ccc|c} 1 \amp 0 \amp 0 \amp -1 \\ 0 \amp 1 \amp 0 \amp 3 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array} \right] \end{equation*}
    🔗
    Solution.
    This is in reduced row echelon form.
    🔗
    🔗
    🔗

    (b)

    \begin{equation*} \left[ \begin{array}{ccc|c} 1 \amp 2 \amp -1 \amp 2 \\ 0 \amp 1 \amp -2 \amp 0 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array} \right] \end{equation*}
    🔗
    Solution.
    This is not in reduced row echelon form because there are nonzero elements in the columns that have leading entries.
    \begin{equation*} \left[ \begin{array}{ccc|c} 1 \amp 2 \amp -1 \amp 2 \\ 0 \amp 1 \amp -2 \amp 0 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array} \right]\sim \left[ \begin{array}{ccc|c} 1 \amp 2 \amp 0 \amp 3 \\ 0 \amp 1 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array} \right]\ \sim\left[ \begin{array}{ccc|c} 1 \amp 0 \amp 0 \amp -1 \\ 0 \amp 1 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array} \right] \end{equation*}
    🔗
    🔗
    🔗

    (c)

    \begin{equation*} \left[ \begin{array}{ccc|c} 2 \amp 0 \amp 4 \amp -8 \\ 0 \amp 1 \amp 3 \amp 2 \\ \end{array} \right] \end{equation*}
    🔗
    Solution.
    We scale the first row by \(1/2\) so that
    \begin{equation*} \left[ \begin{array}{ccc|c} 2 \amp 0 \amp 4 \amp -8 \\ 0 \amp 1 \amp 3 \amp 2 \\ \end{array} \right] \sim \left[ \begin{array}{ccc|c} 1 \amp 0 \amp 2 \amp -4 \\ 0 \amp 1 \amp 3 \amp 2 \\ \end{array} \right] \end{equation*}
    🔗
    🔗
    🔗

    (d)

    \begin{equation*} \left[ \begin{array}{ccc|c} 0 \amp 1 \amp 3 \amp 2 \\ 0 \amp 0 \amp 0 \amp 0 \\ 1 \amp 0 \amp 4 \amp 2 \\ \end{array} \right] \end{equation*}
    🔗
    Solution.
    We can interchange rows so that
    \begin{equation*} \left[ \begin{array}{ccc|c} 0 \amp 1 \amp 3 \amp 2 \\ 0 \amp 0 \amp 0 \amp 0 \\ 1 \amp 0 \amp 4 \amp 2 \\ \end{array} \right] \sim \left[ \begin{array}{ccc|c} 1 \amp 0 \amp 4 \amp 2 \\ 0 \amp 1 \amp 3 \amp 2 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array} \right] \end{equation*}
    🔗
    🔗
    🔗

    (e)

    \begin{equation*} \left[ \begin{array}{ccc|c} 1 \amp 0 \amp 4 \amp 2 \\ 0 \amp 1 \amp 3 \amp 2 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array} \right] \end{equation*}
    🔗
    Solution.
    The last row has a leading entry but there are nonzero elements above it:
    \begin{equation*} \left[ \begin{array}{ccc|c} 1 \amp 0 \amp 4 \amp 2 \\ 0 \amp 1 \amp 3 \amp 2 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array} \right] \sim \left[ \begin{array}{ccc|c} 1 \amp 0 \amp 4 \amp 0 \\ 0 \amp 1 \amp 3 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array} \right] \end{equation*}
    🔗
    🔗
    🔗
    🔗

    7.

    (a)

    Create, if possible, an augmented matrix in reduced row echelon form that represents a linear system having three equations, two variables, and exactly one solution. If this is not possible, state the reason why.
    🔗
    Solution.
    \begin{equation*} \left[ \begin{array}{cc|c} 1 \amp 0 \amp -1 \\ 0 \amp 1 \amp 2 \\ 0 \amp 0 \amp 0 \\ \end{array} \right] \end{equation*}
    🔗
    🔗
    🔗

    (b)

    Create, if possible, an augmented matrix in reduced row echelon form that represents a linear system having four equations, two variables, and no solutions. If this is not possible, state the reason why.
    🔗
    Solution.
    \begin{equation*} \left[ \begin{array}{cc|c} 1 \amp 0 \amp -1 \\ 0 \amp 0 \amp 2 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \\ \end{array} \right] \end{equation*}
    🔗
    🔗
    🔗

    (c)

    Create, if possible, an augmented matrix in reduced row echelon form that represents a linear system having two equations, four variables, and exactly one solution. If this is not possible, state the reason why.
    🔗
    Solution.
    This is not possible since there will always be free variables:
    \begin{equation*} \left[ \begin{array}{cccc|c} 1 \amp 0 \amp * \amp * \amp -1 \\ 0 \amp 1 \amp * \amp * \amp 2 \\ \end{array} \right] \end{equation*}
    🔗
    🔗
    🔗
    🔗

    8.

    (a)

    If an augmented matrix is in reduced row echelon form, what does a row whose entries are all zero tell us about the set of solutions?
    🔗
    Solution.
    It doesn’t tell us anything since any value for the variables will work.
    🔗
    🔗
    🔗

    (b)

    How can you quickly determine if a linear system has no solutions by looking at the reduced row echelon form of its augmented matrix?
    🔗
    Solution.
    If there is a leading entry in the last column, that equation is telling us that \(0=1\text{,}\) which cannot be true. Therefore, there are no solutions.
    🔗
