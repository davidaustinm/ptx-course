    Worksheet Reduced row echelon form
    

    MTH 204: January 20
    🔗
    Remember that a matrix in reduced row echelon form is characterized by these properties:
    1. If the entries in a row are all zero, then the same is true of any row below it.
      🔗
      🔗
    2. If we move across a row from left to right, the first nonzero entry we encounter is 1. We call this entry the leading entry in the row.
      🔗
      🔗
    3. The leading entry in any row is to the right of the leading entries in all the rows above it.
      🔗
      🔗
    4. A leading entry is the only nonzero entry in its column.
      🔗
      🔗
    🔗

    1.

    Consider each of the following augmented matrices. Determine if the matrix is in reduced row echelon form. If it is not, perform a sequence of scaling, interchange, and replacement operations to obtain a row equivalent matrix that is in reduced row echelon form. Then use the reduced row echelon matrix to determine whether there is a single solution, infinitely many solutions, or no solutions. If there are infinitely many, identify free and basic variables.
    🔗

    (a)

    \(\left[ \begin{array}{rrr|r} 1 \amp -1 \amp 2 \amp -2 \\ 0 \amp 1 \amp 0 \amp 3 \\ 0 \amp 0 \amp 1 \amp 1 \\ \end{array} \right]\)
    🔗
    🔗

    (b)

    \(\left[ \begin{array}{rrr|r} 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 3 \amp 2 \\ 1 \amp -2 \amp 2 \amp 8 \\ \end{array} \right]\)
    🔗
    🔗

    (c)

    \(\left[ \begin{array}{rrr|r} 1 \amp 0 \amp 4 \amp 2 \\ 0 \amp 0 \amp -1 \amp 2 \\ 0 \amp 0 \amp 1 \amp -2 \\ \end{array} \right]\)
    🔗
    🔗

    (d)

    \(\left[ \begin{array}{rrr|r} 0 \amp 1 \amp 3 \amp 2 \\ 0 \amp 0 \amp 0 \amp 1 \\ 1 \amp 0 \amp 4 \amp 2 \\ \end{array} \right]\)
    🔗

    🔗
    🔗

    (c)

    3.

    (a)

    If a matrix is in reduced row echelon form, remember that the leading entries are the leftmost ones in a row. We refer to the locations where the leading entries occur as pivot positions.
    🔗
    Circle the pivot positions in the reduced row echelon form below:
    \begin{equation*} \left[ \begin{array}{rrrr|r} 1 \amp 3 \amp 0 \amp 2 \amp 0 \\ 0 \amp 0 \amp 1 \amp -1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \\ \end{array} \right]\text{.} \end{equation*}
    🔗
    🔗

    (b)

    If a system of linear equations has no solutions, what can you say about the pivot positions?
    🔗
    🔗

    (c)

    If a system of linear equations has infinitely many solutions, what can you say about the pivot positions?
    🔗
    🔗

    (d)

    If a system of linear equations has exactly one solution, what can you say about the pivot positions?
    🔗
    🔗

    4.

    There is a page of Sage cells at gvsu.edu/s/0Ng. You will be using this frequently.
    1. To define a matrix, use
      matrix([[1,2,3,4],[5,6,7,8]])
      🔗
      🔗
    2. You can give a matrix a name:
      A = matrix([[1,2,3,4],[5,6,7,8]])
      🔗
      🔗
    3. To find the reduced row echelon form,
      matrix([[1,2,3,4],[5,6,7,8]]).rref()
      or
      A.rref()
      🔗
      🔗

    The coefficient matrix is the portion of the augmented matrix to the left of the equal sign. For instance, the coefficient matrix of the augmented matrix above is
    \begin{equation*} \left[ \begin{array}{ccc} -2 \amp 3 \amp 0 \\ -1 \amp 4 \amp 1 \\ 3 \amp 0 \amp 2 \\ 1 \amp 5 \amp 3 \\ \end{array} \right]\text{.} \end{equation*}
